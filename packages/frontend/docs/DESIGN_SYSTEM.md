# デザインシステム - 高級感モダン

このドキュメントでは、アプリケーション全体で統一する「高級感のあるモダン」なデザインの指針を定義します。

## コンセプト

- **高級感**: 落ち着いた配色、十分な余白、洗練されたタイポグラフィ
- **モダン**: シャープなライン、明確な視覚的階層、ミニマルなアイコン使用
- **統一感**: 全ページで同じパターンを繰り返し使用
- **統一borderRadius**: 全コンポーネントで4px（シャープな印象）

| 用途 | 色 | 値 |
|-----|---|---|
| プライマリ（アクセント） | 紺色 | `#1e3a5f` |

| 背景（メインコンテンツ） | 明るいグレー | `#fafafa` |
| 背景（サブコンテンツ） | グレー | `#f1f5f9` |
| 背景（カード） | 淡いグレー | `#f8fafc` |
| テキスト（主） | ダークグレー | `#1e293b` |
| テキスト（副） | グレー | `#64748b` |
| テキスト（薄い） | ライトグレー | `#94a3b8` |

### 禁止する色

| 色 | 理由 |
|---|-----|
| `#ffffff`（真っ白） | 眩しく、安っぽく見える |
| `#000000`（真っ黒） | 硬すぎる |
| `#ff0000`, `#00ff00`, `#0000ff`（原色） | 安っぽく見える |

## タイポグラフィ

### フォントサイズ階層

| 用途 | サイズ | ウェイト | 備考 |
|-----|-------|---------|-----|
| ページタイトル | 1.1rem | 700 | |
| セクションタイトル | 0.9rem | 600 | 左アクセントライン付き |
| カードタイトル | 0.85rem | 600 | |
| 本文 | 0.85rem | 400 | |
| 補助テキスト/ラベル | 0.75rem | 500 | |

### ルール

- 文字間隔: `0.02em`
- 行間: 1.5〜1.6
- フォントサイズは**4段階まで**に抑える（多すぎると雑多になる）

---

## スペーシング

| 用途 | 値 | px換算 |
|-----|---|-------|
| セクション間 | 3 | 24px |
| カード内パディング | 2〜2.5 | 16〜20px |
| 要素間 | 1.5〜2 | 12〜16px |
| 小要素間 | 1 | 8px |

### ルール

- 余白は**広めに**とる（詰め込まない）
- 呼吸感を大切にする

## コンポーネントスタイル

### カード

```
背景: #f8fafc
角丸: 4px（統一・シャープに）
境界線: 1px solid #e2e8f0
影: なし or 軽め（0 1px 3px rgba(0,0,0,0.06)）
パディング: 16px (2)
```

**オプション**: 左にアクセントライン（3px、ステータス色）

### セクションタイトル

```
左アクセントライン: 3px、青グラデーション
  background: linear-gradient(180deg, #3b82f6 0%, #2563eb 100%)
色: #1e3a5f（紺色）
サイズ: 0.85〜0.9rem
ウェイト: 600
アイコン: 使わない
```

### 境界線

| 用途 | スタイル |
|-----|---------|
| 通常の区切り | `1px solid #e2e8f0` |
| 強調（ヘッダー下など） | `2px solid #cbd5e1` |

### 影

| 用途 | スタイル |
|-----|---------|
| カード | なし or `0 1px 3px rgba(0,0,0,0.06)` |
| 浮遊要素 | `0 2px 8px rgba(0,0,0,0.08)` |
| ヘッダー下 | `0 1px 2px rgba(0,0,0,0.05)` |

### borderRadius（統一: 4px）

```typescript
// 全コンポーネントで統一
borderRadius: borderRadius.xs  // '4px'
```

**重要**: 角丸は全て4pxに統一し、シャープでモダンな印象を与える。

## アイコン使用ルール

### 原則

- **控えめに使う** - 必要な箇所のみ
- **タイトルの横には置かない** - 左アクセントラインで代用
- サイズ: 16〜18px
- 色: `#64748b`（グレー）

### 使用例

| OK | NG |
|---|---|
| 連絡先の横にアイコン | 全ての項目にアイコン |
| ステータス表示 | 装飾目的のアイコン |

## レイアウトルール

### 視覚的階層

1. **プライマリ**: ユーザーが最初に見るべき情報
2. **セカンダリ**: 補足情報
3. **ターシャリ**: 詳細・補助情報

階層を**明確に**分ける。

### 情報密度

- 詰め込まない
- 1つのカード/セクションに情報を入れすぎない
- 必要に応じて折りたたみを使用

## チェックリスト

デザイン実装時に確認すること：

- [ ] 真っ白（#ffffff）を使っていないか
- [ ] 原色を使っていないか
- [ ] 余白は十分か
- [ ] フォントサイズ階層は4段階以内か
- [ ] アイコンを使いすぎていないか
- [ ] セクションタイトルに左アクセントラインがあるか
- [ ] カードの角丸は4px（borderRadius.xs）か
- [ ] 視覚的階層は明確か
- [ ] 一覧ページはカード形式+右パネルレイアウトか
